# PWA icons

Replace these placeholders with real PNG assets before deploying:

- `icon-192.png` — 192×192 PNG, transparent or accent background
- `icon-512.png` — 512×512 PNG, supports `purpose: any maskable`
- `apple-touch-icon.png` — 180×180 PNG

You can generate them from a single source SVG with a tool like
[`pwa-asset-generator`](https://github.com/elegantapp/pwa-asset-generator).
